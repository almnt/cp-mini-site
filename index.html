Crimson Pro

is a variable font

for book and digital production in the tradition of beautiful oldstyle typefaces.

Contemporary and classic, Crimson Pro comes in eight pre-defined weights from Extra light to Black, in Roman and Italic.

The Benefits of Crimson Pro & Variable Fonts

Variable Font

Instead of multiple font files (one for Bold, one for Medium, another for Light), with Crimson Pro, you get one compressed font file thatcontains all weights.

While Crimson Pro currently only has avariable Weight (wght), a variable font can also allow forchanging other properties, such as Optical size (opsz), Slant (slnt), Width (wdth), and Italic (ital).

What does this mean?

One font file means better performance and fast loading times.
